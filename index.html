Scalable Vector Graphics (SVG)

  • XML-based markup language
  • defined in XML text files
  • open web standard for images (W3C, since 1999)
  • optimized to integrate with other web standards
  • used a lot in web design for things like icons, logos, animations, and interactive elements

Coordinates

All SVG content is drawn inside SVG viewports. Every SVG viewport defines a drawing region characterized by size (width, height), and an origin, measured in abstract user units. (W3C)
